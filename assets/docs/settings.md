## Bilgecan â€“ Settings

The Settings page allows users to manage their personal preferences in Bilgecan.  
It is focused on essential configuration only, keeping the experience simple and secure.

### Available Settings

#### **Preferred Locale**
Users can select their preferred language/locale for the Bilgecan user interface.

English and Turkish languages are available.

**Details:**
- Affects UI texts and messages
- Applied immediately after saving preference
- Stored per user 

#### **Password Reset**
Users can securely change their account password.

**Details:**
- Requires the current password
- New password is validated before saving
- Passwords are stored securely using hashing
